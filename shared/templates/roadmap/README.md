# Roadmap Templates

Copie esta pasta para `<projeto>/roadmap/` ao iniciar um novo projeto.

## Uso
```bash
cp -r ~/.claude/shared/templates/roadmap/ ./roadmap/
```

## Personalizacao
Apos copiar, editar `README.md` do projeto para definir:
- Lista de MODULOS especificos do projeto
- Convencoes adicionais

## Arquivos
- `backlog.template.md` — Template de backlog unificado
- `sprint.template.md` — Template de sprint semanal
- `prd.template.md` — Template de PRD (Product Requirements Document)
- `spec.template.md` — Template de SPEC (Technical Specification)
- `bug.template.md` — Template de bug report
- `feature.template.md` — Template de feature request
