## Metodologia PRISM-Lite

```
P — Perceber (campo, entrevistas, observação)
R — Revelar (análise de dados, métricas)
I — Investigar (causa raiz, 5 porquês)
S — Solucionar (proposta, priorização)
S2— Pesquisa Profunda (benchmarks, alternativas)
M — Manter (monitoramento, KPIs)
D — Detalhar (implementação, playbook)
```
