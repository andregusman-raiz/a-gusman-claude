# Finnet — Plataforma Financeira

> Fonte: https://finnet.com.br/
> API Universal: https://finnet.com.br/api-universal/
> Atualizado: 2026-03-21

## Visao Geral

Finnet é a plataforma de gestão financeira e automação de processos bancários.
Conecta ERPs a múltiplas instituições financeiras via API unificada.

## Produtos

### API Universal
- Interface unificada para comunicação com múltiplos bancos
- Dados em tempo real: saldos, extratos, pagamentos, confirmações
- Padronização de formato de dados entre instituições
- Integração multibanco via conexão única

### EDI Financeiro
- Tráfego de arquivos bancários (CNAB)
- Conversão CNAB → API e API → CNAB
- Processo completo API → API entre sistemas

### Funcionalidades
- Monitoramento de caixa em tempo real
- Processamento automático de pagamentos
- Consolidação e análise de dados financeiros
- Compliance e auditoria
- Integração com ERPs (TOTVS RM, etc.)

### Segurança
- Protocolos de autenticação
- Criptografia de dados
- Controle de acesso
- Instituição de Pagamento com infraestrutura própria

## Open Finance (2025-2026)
- APIs homogêneas e previsíveis
- Integrações Open Finance aprimoradas
- Automação de pagamentos via Pix
- Camada de segurança fortalecida
