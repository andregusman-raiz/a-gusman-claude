# Finnet API Universal

> Source: https://finnet.com.br/api-universal/ e https://finnet.com.br/api/

## Conceito

A API Universal e uma interface de integracao que permite a comunicacao entre diferentes sistemas financeiros, conectando dados e servicos de multiplas fontes em um padrao unico. Funciona como uma "ponte inteligente entre o sistema da empresa e sistemas externos."

## Funcionalidades Principais

### Integracao Centralizada
- Conecta bancos, meios de pagamento e plataformas em um unico fluxo
- Elimina necessidade de construir conexoes individuais para cada parceiro
- Abstrai layouts bancarios complexos

### Conversao de Formatos
- **CNAB para API**: Converter arquivos bancarios tradicionais em interfaces modernas
- **API para CNAB**: Processar dados via API e enviar aos bancos em formato legado
- **API para API**: Comunicacao direta entre sistemas modernos

### Operacoes Suportadas
- Registro e consulta de lancamentos de extrato
- Consulta de titulos e saldos bancarios
- Envio e consulta de pagamentos
- Emissao de comprovantes
- Geracao de boletos nao registrados
- Rota para controle de NSA
- Saldos bancarios
- Extratos e movimentacoes
- Informacoes de pagamentos e confirmacoes
- Consumo estruturado de dados em tempo real

### ITP (Iniciadora de Transacao de Pagamento)
- Conecta-se via API ao diretorio do BACEN
- Agiliza troca de informacoes e transacoes
- Garante seguranca operacional

## Processo de Funcionamento

1. Interpreta solicitacoes
2. Direciona comunicacao para destino correto
3. Retorna resposta padronizada
4. Aplica autenticacao e seguranca
