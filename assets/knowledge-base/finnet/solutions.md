## Categorias de Solucoes

### 1. Credito

#### Painel Fornecedor
- Supply Chain Finance: gestao unificada de notas fiscais de fornecedores
- Criacao de linhas de credito atraves do Risco Sacado
- Gestao centralizada de documentos fiscais
- Plataforma para negociacao com instituicoes financeiras
- Resultado: cliente relatou aumento de "45dd" no prazo medio com fornecedores

#### Antecipag
- Solucao de antecipacao de recebiveis

### 2. Contas a Pagar / Tesouraria

#### Bankmanager
- Internet Banking Multibanco e Multiempresa
- Tres modulos: Pagamentos, Extratos, Cobranças
- Otimiza rotina do departamento financeiro
- "Reduziu drasticamente o tempo de processamento, aprovação e controle das movimentações financeiras"

### 3. Cobranca e Recebimentos

#### Luna (Luna & Bolepix)
- Portal de Boletos integrado
- Automatizacao do processo de cobranca
- Multiplas formas de pagamento em unico lugar
- Reducao de inadimplencia
- Acelera recebimentos

### 4. Conectividade Financeira

#### EDI Financeiro
- Automatizacao de envio de arquivos bancarios
- Cobertura: todos os bancos utilizados pela empresa
- Agilidade e seguranca em trafego de dados

#### Open Finance
- Acesso a solucoes otimizadas do ecossistema Open Finance

#### APIs de Integracao
- Conectividade programatica com sistemas financeiros
- API Universal, API Multibanco

### 5. Solucoes Customizadas

#### Para Bancos
- Solucoes especificas para instituicoes financeiras

#### Outras Solucoes
- Ofertas especializadas sob demanda
