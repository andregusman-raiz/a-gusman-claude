Spawn the `ag-15-auditar-codigo` agent as a background subagent to perform a security audit.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-15-auditar-codigo`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Scope: [full | specific module/route]
Focus: [OWASP | RLS | secrets | LGPD | all — default: all]

[user instructions from $ARGUMENTS]

Execute auditoria de seguranca. Verifique RLS, secrets expostos, OWASP top 10, e LGPD compliance.
```

## Important

- Do NOT try to audit inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with security findings by severity
