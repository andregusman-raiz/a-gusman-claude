Spawn the `ag-Q-22-testar-e2e` agent as a background subagent to run E2E tests with Playwright.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-Q-22-testar-e2e`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Modo: [gerar | executar | gap-analysis | assertion-hardening]
Scope: [fluxos ou modulos a testar]

[user instructions from $ARGUMENTS]

Execute os testes E2E seguindo as instrucoes. Comece verificando pre-requisitos (Playwright instalado, app rodando).
```

## Important

- Do NOT try to run E2E tests inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
