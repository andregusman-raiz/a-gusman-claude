Use the ag-37-gerar-testes-mcp skill to generate Playwright tests from real browser flows.

This is a **skill** (not a subagent) — it runs inline in the current session using Playwright MCP.

## User arguments

$ARGUMENTS

## What it does

1. Connects to Playwright MCP browser
2. Records user flows (navigation, clicks, form fills)
3. Generates .spec.ts test files from recorded flows
4. Saves tests to the project's test directory

## Requirements

- Playwright MCP must be available in the session
- A running application URL to test against
