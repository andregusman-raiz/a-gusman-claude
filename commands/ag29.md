Spawn the `ag-W-29-gerar-documentos` agent as a background subagent to generate Office documents (PDF, DOCX, XLSX, PPTX).

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-W-29-gerar-documentos`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Tipo: [pdf | docx | xlsx | pptx]
Output: [output file path]

[user instructions from $ARGUMENTS]

Gere o documento conforme especificado. Use as ferramentas apropriadas para o formato.
```

## Important

- Do NOT try to generate documents inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with the output file path
