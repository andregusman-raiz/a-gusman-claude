Spawn the `ag-09-depurar-erro` agent as a background subagent to debug and fix errors using root-cause analysis.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-09-depurar-erro`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Erro: [error message or description]
Contexto: [stack trace, logs, or reproduction steps]

[user instructions from $ARGUMENTS]

Depure usando Root Cause Analysis. Trace a cadeia completa antes de implementar fix. Max 2 tentativas.
```

## Important

- Do NOT try to debug inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with the root cause and fix applied
