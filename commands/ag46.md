Spawn the `ag-X-46-buscar-voos` agent to search for real-time flight prices via Google Flights.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-X-46-buscar-voos`
- `mode`: `auto`
- `prompt`: Pass the user's arguments below

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Busca de voos solicitada:

[user instructions from $ARGUMENTS]

Execute a busca usando fast-flights (Google Flights). Apresente resultados em tabela Markdown ordenada por horario.
```

## Important

- Do NOT try to search flights inline — ALWAYS spawn as Agent subagent
- The agent uses `fast-flights` Python library (installs automatically if needed)
- Data comes from Google Flights — real prices, not estimates
- After spawning, confirm to the user that the search is running
