Spawn the `ag-14-criticar-projeto` agent as a background subagent to perform code review on a PR or branch.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-14-criticar-projeto`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
PR: [PR number or branch name to review]
Focus: [security | performance | quality | all — default: all]

[user instructions from $ARGUMENTS]

Execute code review completo. Verifique quality gates, anti-patterns, e test coverage.
```

## Important

- Do NOT try to review code inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with review findings
