Spawn the `ag-28-saude-sessao` agent as a background subagent to check session health and context pressure.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-28-saude-sessao`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Check: [memory | context | git | all — default: all]

[user instructions from $ARGUMENTS]

Verifique saude da sessao: memory_pressure, context window usage, git status, processos pendentes.
```

## Important

- Do NOT try to check health inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with health status and recommendations
