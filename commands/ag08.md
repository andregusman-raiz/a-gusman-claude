Spawn the `ag-08-construir-codigo` agent as a background subagent to build and implement code from a plan.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-08-construir-codigo`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Branch: [current branch or feat/ branch to create]
Scope: [files/modules to implement]

[user instructions from $ARGUMENTS]

Implemente o codigo seguindo o plano. Commits incrementais a cada 3-5 mudancas. Comece verificando branch e git status.
```

## Important

- Do NOT try to implement code inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
