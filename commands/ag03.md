Spawn the `ag-03-explorar-codigo` agent as a background subagent to explore and map the codebase structure.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-03-explorar-codigo`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Focus: [directory, module, or pattern to explore]

[user instructions from $ARGUMENTS]

Explore a codebase e mapeie a estrutura. Documente dependencias entre modulos no Knowledge Graph.
```

## Important

- Do NOT try to explore inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with the codebase map
