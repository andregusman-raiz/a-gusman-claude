Spawn the `ag-11-otimizar-codigo` agent as a background subagent to optimize performance.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-11-otimizar-codigo`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `isolation`: `worktree`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Scope: [files/modules to optimize]
Objetivo: [bundle size, runtime performance, memory, load time]

[user instructions from $ARGUMENTS]

Otimize o codigo seguindo as instrucoes. Cada passo com commit. Comece verificando branch e git status.
```

## Important

- Do NOT try to optimize inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
