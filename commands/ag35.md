Spawn the `ag-35-incorporar-modulo` agent as a background subagent to execute module incorporation.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-35-incorporar-modulo`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `isolation`: `worktree`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Modulo externo: [path or repo of module to incorporate]
Estrategia: [strangler fig, ACL, rewrite]

[user instructions from $ARGUMENTS]

Execute a incorporacao seguindo o roadmap do ag-34. Cada fase com commit. Comece verificando branch e git status.
```

## Important

- Do NOT try to incorporate inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
