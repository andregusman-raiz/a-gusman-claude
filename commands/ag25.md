Spawn the `ag-B-25-diagnosticar-bugs` agent as a background subagent to triage and diagnose bugs.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-B-25-diagnosticar-bugs`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Bugs: [list of bugs, issue numbers, or error descriptions]

[user instructions from $ARGUMENTS]

Faca triagem e diagnostico de cada bug. Classifique por severidade (P0-P3) e identifique causa raiz provavel.
```

## Important

- Do NOT try to diagnose inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with triage report per bug
