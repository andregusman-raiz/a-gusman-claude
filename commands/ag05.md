Spawn the `ag-05-pesquisar-referencia` agent as a background subagent to research references, alternatives, and best practices.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-05-pesquisar-referencia`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Topico: [technology, pattern, or problem to research]

[user instructions from $ARGUMENTS]

Pesquise referencias, alternativas, e best practices. Apresente comparativo com pros/cons.
```

## Important

- Do NOT try to research inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with research findings
