Spawn the `ag-10-refatorar-codigo` agent as a background subagent to refactor code structure.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-10-refatorar-codigo`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `isolation`: `worktree`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Scope: [files/modules to refactor]
Objetivo: [extract, rename, reorganize]

[user instructions from $ARGUMENTS]

Refatore o codigo seguindo as instrucoes. Cada passo com commit. Comece verificando branch e git status.
```

## Important

- Do NOT try to refactor inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
