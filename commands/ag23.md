Spawn the `ag-23-bugfix-batch` agent as a background subagent to fix bugs in sequential batches of 3-5.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-23-bugfix-batch`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Bugs: [list of bug IDs, file paths, or descriptions]
Batch size: [3-5 — default: 3]

[user instructions from $ARGUMENTS]

Corrija os bugs em batches sequenciais. Commit apos cada batch. Typecheck + lint antes de cada commit.
```

## Important

- Do NOT try to fix bugs inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with fix summary per batch
