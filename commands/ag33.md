Spawn the `ag-33-mapear-integracao` agent as a background subagent to map integration dimensions between systems.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-33-mapear-integracao`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Source: [system/module being integrated]
Target: [destination system/module]

[user instructions from $ARGUMENTS]

Mapeie dimensoes de integracao: API surface, data models, auth, error handling, e migration path.
```

## Important

- Do NOT try to map integrations inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with integration map and risk assessment
