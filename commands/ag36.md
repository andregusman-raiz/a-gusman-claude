Spawn the `ag-36-testar-manual-mcp` agent as a background subagent to run exploratory QA via Playwright MCP.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-36-testar-manual-mcp`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
URL: [application URL to test]
Fluxos: [user flows to explore]

[user instructions from $ARGUMENTS]

Execute testes exploratorios via Playwright MCP. Documente bugs encontrados com screenshots e steps to reproduce.
```

## Important

- Do NOT try to run exploratory QA inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete
