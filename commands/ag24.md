Spawn the `ag-24-bugfix-paralelo` agent as a background subagent to fix 6+ bugs in parallel using Agent Teams.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-24-bugfix-paralelo`
- `mode`: `bypassPermissions`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Bugs: [list of 6+ bug IDs, file paths, or descriptions]
Max teammates: [2-4 — default: 3]

[user instructions from $ARGUMENTS]

Divida os bugs em grupos independentes (sem file overlap). Spawne teammates em paralelo. Merge ao final.
```

## Important

- Do NOT try to fix bugs inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with per-teammate results and merge status
