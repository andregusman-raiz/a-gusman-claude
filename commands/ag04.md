Spawn the `ag-04-analisar-contexto` agent as a background subagent to analyze context, dependencies, and tech debt.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-04-analisar-contexto`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Focus: [specific area or "full analysis"]

[user instructions from $ARGUMENTS]

Analise o contexto completo: stack, dependencias, tech debt, e riscos. Registre findings no Knowledge Graph.
```

## Important

- Do NOT try to analyze inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with analysis report
