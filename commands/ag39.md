Spawn the `ag-39-ciclo-teste-completo` agent as a background subagent to run the full test cycle autonomously.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-39-ciclo-teste-completo`
- `mode`: `bypassPermissions` (agent roda autonomo sem perguntas)
- `run_in_background`: `true` (ciclo longo, nao bloquear sessao)
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Scope: [full | jest-only | vitest-only | playwright-only — default: full]
Max cycles: [1-3 — default: 3]

[any extra user instructions from $ARGUMENTS]

Execute o ciclo completo Test-Fix-Retest seguindo suas instrucoes. Comece pela Phase 0 (preflight).
```

## Important

- Do NOT try to execute the test cycle inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with the final report summary
