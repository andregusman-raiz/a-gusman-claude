Spawn the `ag-Q-12-validar-execucao` agent as a background subagent to validate implementation completeness against spec.

## What to do

Use the **Agent tool** with these parameters:
- `subagent_type`: `ag-Q-12-validar-execucao`
- `mode`: `auto`
- `run_in_background`: `true`
- `prompt`: Pass the user's arguments below plus the current working directory

## User arguments

$ARGUMENTS

## Prompt template

Compose the Agent prompt like this:

```
Projeto: [CWD or user-provided path]
Spec: [path to SPEC.md or task_plan.md]

[user instructions from $ARGUMENTS]

Valide item por item contra o spec. Reporte: Total X | Completos Y | Parciais Z | Faltando W.
```

## Important

- Do NOT try to validate inline — ALWAYS spawn as Agent subagent
- After spawning, confirm to the user that the agent is running in background
- The agent will notify when complete with validation checklist
